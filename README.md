## word2vec
本程序使用Python3.6和Numpy实现了基于Negative Sampling的Skip-gram词向量模型，并且集成了Subsampling技术，同时还包括对低频词的裁剪工作；最后利用余弦相似，求得与输入词汇最相似的10个词汇，来检验词向量模型训练的效果。
